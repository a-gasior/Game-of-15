Otis Puzzle

Game Tutorial

If you've never played before, try the simple version. You can do that by clicking the button below. The goal of the game is to organize all the numbers from left to right top to bottom starting at zero. Items may only be moved if an adjacent square is empty.

If you want to reinitialize the map you can click the New Game button below. It will reconfigure a new map for you.
